- All classes and library files must be on **src/base** folder.
- Samples must be save on equivalent FILE_LIBRARY (E.g **world_library.js**).
- Generators must be on **THE END OF THE FILE** that contains the classes.
#### Project File Format
```js ignore
- index.html
- main.js
- style.css
- src
	- engine.js
	- base
		- ...
```
